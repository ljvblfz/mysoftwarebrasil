    [Tutorial] Batch Script

    Ol&#225;, irei explicar aqui algumas coisinhas sobre batch script, exemplos, n&#227;o apenas para Win98, mas tamb&#233;m para NT/XP pois existem poucos documentos para XP em portugu&#234;s...

    Sei que j&#225; tem alguns materiais sobre batch script, se acharem desnecess&#225;rio, podem desconsiderar/deletar o post.

    O que &#233; um BAT?
    Tipo de arquivo em lote (batch file) do DOS e sistemas Windows. Neste tipo de arquivo pode-se entrar v&#225;rios comandos de linha de comando a serem executados, em vez de ter de escrev&#234;-los um a um na linha de comando. O arquivo em lote com extens&#227;o BAT mais famoso &#233; o Autoexec.bat, que &#233; executado toda a vez em que o sistema operacional &#233; carregado.

    Bom, vamos conhecer agora alguns comandos que iremos usar frequentemente, se tiver d&#250;vida, digite COMANDO /? que ir&#225; ser mostrado uma explica&#231;&#227;o e exemplos do seu uso:

    ECHO
    Serve para que seja ativado ou desativado o "eco" do comando ou para que seja mostrado uma mensagem. Usa-se echo. para quebrar uma linha.

    Exemplo1:
    Código:
    @echo off
echo Ol&#225; mundo!
echo.
echo :D
    O @ antes do echo off serve para que o mesmo n&#227;o seja mostrado, e assim ele desative o "eco" silenciosamente. Se tiver alguma d&#250;vida, crie um exemplo sem usar o @.

    Exemplo 2:
    Código:
    @echo off
echo Inicio do arquivo > Teste.txt
echo. >> Teste.txt
echo Bla bla bla >> Teste.txt
    Isso far&#225; com que seja escrito os dados no arquivo Teste.txt, ficando assim:
    Inicio do arquivo

    Bla bla bla

    CLS
    Serve para limpar a tela.. &#233; como se fosse um /clear para quem usa IRC.. ^^

    GOTO
    Serve para alterar a ordem normal de execu&#231;&#227;o do batch, fazendo ir para um r&#243;tulo espec&#237;fico, um r&#243;tulo &#233; identificado com o :.

    Exemplo:
    Código:
    @echo off 
GOTO UIA
echo Deveria aparecer isso se n&#227;o fosse o r&#243;tulo :D
:UIA
echo Pulamos para c&#225;! hehehe
    Par&#226;metros
    Isso &#233; muito interessante.. &#233; para que se possa usar par&#226;metros de execu&#231;&#227;o em seu batch. Em programas por exemplo que precisa ser digitado:
    nomedoprograma /i (para instalar)
    nomedoprograma /d (para desintalar)
    ou pode ser -i ao inv&#233;s de /i e -d ao inv&#233;s de /d.
    No batch script, pode-se identificar o par&#226;metro, atrav&#233;s disso:
    %1 - Primeiro par&#226;metro
    %2 - Segundo par&#226;metro e assim sucessivamente.

    Exemplo:
    Código:
    @echo off
echo %1 - Parametro 1
echo %2 - Parametro 2
    Execute-o assim: nomedoarquivo vamos ver
    Aparecer&#225;:
    vamos - Parametro 1
    ver - Parametro 2.

    Simples n&#233;?

    SET
    Serve para exibir, definir ou remover vari&#225;veis.
    Vari&#225;vel &#233; algo que guarda alguma informa&#231;&#227;o, se chama vari&#225;vel por que seu valor pode variar. (duh)
    Para setar uma vari&#225;vel: SET variavel=seu_valor
    Para visualizar ou usar o valor da vari&#225;vel digite: %variavel%.

    Exemplo:
    Código:
    @echo off
set nome=""
Digite o seu nome:
set /p nome=
echo.
echo Ola %nome% !!
    IF / ELSE
    &#201; usado em praticamente todas linguagens de programa&#231;&#227;o, serve para impor uma condi&#231;&#227;o.

    Código:
    @echo off
set opcao=""
echo Digite uma opcao (1 ou 2):
set /p opcao=
IF [%opcao%] EQU [1] (GOTO OPCAO1) ELSE (GOTO OPCAO2)
:OPCAO1
echo Opcao 1 bla bla bla
goto FIM
:OPCAO2
echo Opcao 2 bla bla bla
:FIM
echo CABOU!
    Se tiver d&#250;vidas sobre o IF, digite no prompt IF /?

    CHOICE (Em Win9x)
    Possibilita dar op&#231;&#245;es de escolha para a pessoa.

    Exemplo:
    Código:
    @echo off
choice /n/c123
if errorlevel 1 goto A1
if errorlevel 2 goto A2
if errorlevel 3 goto A3
:A1
echo Acao 1
goto FIM
:A2
echo Acao 2
goto FIM
:A3
echo Acao 3
goto FIM
:FIM
echo Kabou..
    Bom, visto algumas fun&#231;&#245;es, d&#234; uma re-lida, use o par&#226;metro /? para conhecer mais sobre cada comando, crie coisinhas bobas para ver se j&#225; entendeu..

    Bom, iremos criar duas tools(ferramentas), a primeira &#233; para o comando PING e outra ser&#225; para quem usa rede, pra facilitar o uso do NET SEND.

    Talvez possa ter incompatibilidade com Win9x j&#225; que irei fazer no XP..

    Muitos acham que o PING &#233; apenas ping IP/HOST, mas desconhecem suas outras op&#231;&#245;es, ent&#227;o vamos l&#225;, d&#234; um PING /?:
    Código:
    Uso: ping [-t] [-a] [-n num] [-l tamanho] [-f] [-i TTL] [-v TOS]
          [-r num] [-s num] [[-j lista_hosts] | [-k lista_hosts]]
          [-w tempo_limite] nome_destino

Op&#231;&#245;es:
    -t              Dispara contra o host especificado at&#233; ser interrompido.
                    Para ver estat&#237;sticas e continuar, pressione CTRL-Break;
                    para terminar, pressione CRTL-C.
    -a              Resolve endere&#231;os para nomes de host.
    -n num          N&#250;mero de requisi&#231;&#245;es de eco a enviar. O valor padr&#227;o &#233; 4.
    -l tamanho      Envia o tamanho do buffer.
    -f              Ativa o sinalizador de n&#227;o-fragmenta&#231;&#227;o no pacote.
    -i TTL          Define o tempo de vida.
    -v TOS          Define o tipo de servi&#231;o.
    -r num          Rota dos pacotes para <num> saltos.
    -s num          Data e hora para <num> saltos.
    -j lista_hosts  Rota ampliada de origens definida em <lista_hosts>.
    -k lista_hosts  Rota restrita de origens definida em <lista_hosts>.
    -w tempo_limite Tempo limite em milissegundos a aguardar para cada resposta.
    &#211;timo, vamos criar uma ferramenta para que ao usu&#225;rio digitar o nome do batch file, ele pergunte o IP, n&#250;mero de requisi&#231;&#245;es. tamanho do pacote e tempo limite.

    M&#227;os a obra:
    D&#234; um: edit xping.bat
    Abrirar o editor.. se n&#227;o gostar, fa&#231;a no bloco de notas mesmo :P
    Obs: REM &#233; usado para comentar uma linha...

    Código:
    @echo off
REM Limpa as variaveis...
set ip=""
set num=""
set pack=""
set timeout=""
echo.
echo Qual o IP desejado?
REM Ira pegar o valor que for digitado e atribuir a variavel ip
set /p ip=
echo.
echo Qual o tamanho do pacote a ser enviado? (max: 65500)
REM Ira pegar o valor que for digitado e atribuir a variavel pack
set /p pack=
echo.
echo Qual sera o n&#250;mero de requisicoes? Coloque 0 para ser infinito.
REM Ira pegar o valor que for digitado e atribuir a variavel num
set /p num=
echo.
echo Qual sera o tempo limite de espera? Em milisegundos.
REM Ira pegar o valor que for digitado e atribuir a variavel timeout
set /p timeout=
echo.
cls
echo **********************************************
echo *                                            *
echo *            XPing por whit3_sh4rk           *
echo *                                            *
echo **********************************************
echo.
REM Mostra qual foi o alvo definido anteriormente, usando a variavel ip
echo Iniciando o ping ao alvo: %ip%
echo.
REM Lembra do numero de requisicoes? (Coloque 0 para ser infinito.)
REM Iremos pegar o valor atribuido ao numero de requisicoes
REM se for igual a 0 ele fara o ping infinito, usando o parametro -t
REM se n&#227;o for igual a 0 ele pega o numero de requisicoes atribuido
REM a variavel %num% e usa-o...
IF [%num%] EQU [0] (
ping -t %ip% -l %pack% -w %timeout%
) ELSE (
ping %ip% -n %num% -l %pack% -w %timeout%
echo.
echo Fim do ping!
)
    &#211;timo, terminanos! Salve isso &#233; claro!
    Agora teste o programa, e tente entender cada linha(nossaaa q dif&#237;cil!!) kkkk depois disso vamos agora para a pr&#243;xima tool, nela usaremos par&#226;metros..
    Obs: S&#243; uma dica, se quiserem usarem par&#226;metros tamb&#233;m nesse "xping" &#233; s&#243; definir corretamente o parametro 1=ip, parametro 2=pack, parametro 3=num, parametro 4=timeout, ae &#233; s&#243; a pessoa usar:
    xping 127.0.0.1 65500 5 1000

    Ou seja, ir&#225; pingar no 127.0.0.1, com um buffer de 65500 bytes, fazer isso 5 vezes, e esperar 1seg pelo resposta...

    M&#227;os a obra 2 a revolta(lol):
    Bom, imaginemos que aqui na rede tem 3 m&#225;quinas, seus nomes:
    LOCKE
    SAWYER
    HURLEY
    (qualquer semelhan&#231;a com o seriado LOST &#233; mera coincid&#234;ncia : )

    D&#234; um: edit msg.bat
    Vamos l&#225;:
    Código:
    @echo off
set msg=""
IF [%1] EQU [] GOTO AFF
IF [%1] EQU [LOCKE] (GOTO LOCKE) ELSE IF [%1] EQU [SAWYER] (GOTO SAWYER) ELSE IF [%1] EQU [HURLEY] (GOTO HURLEY) ELSE (GOTO AFF)
:LOCKE
echo Digite a mensagem que deseja enviar para LOCKE:
set /p msg=
echo.
net send LOCKE %msg%
GOTO :EOF
:SAWYER
echo Digite a mensagem que deseja enviar para SAWYER:
set /p msg=
echo.
net send SAWYER %msg%
GOTO :EOF
:HURLEY
echo Digite a mensagem que deseja enviar para HURLEY:
set /p msg=
echo.
net send HURLEY %msg%
:AFF
echo.
echo Nenhum parametro valido foi especificado!
    Salve..

    Como ser&#225; usado?
    Abra o prompt e digite:
    msg LOCKE
    ou
    msg SAWYER
    ou
    msg HURLEY
