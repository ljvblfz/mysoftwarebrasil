ARQUIVO COM OS PARAMETRO DA LINHA DE COMANDO

database - Nome da base de dados
server - Nome do Servidor
username - Nome do usuario para acessar o banco
password - Senha de acesso ao banco		 
roteiro - Sequencia do roteiro a ser carregado
referencia - Código da referencia		 
target - Caminho e nome do arquivo de saida para carga ou de entrada da descarga 
cliente - Nome do cliente (Strategos, Diadema, Scs). Padrao: Strategos
descarga - Indica que deve ser feita a descarga do arquivo especificado de 'target'
visivel - Indica se a janela de log deve ficar visivel
empty - Cria a base SDF vazia
